Unshield
========

Tool to extract InstallShield .cab files

Dictionary
----------

InstallShield (IS): see www.installshield.com

InstallShield Cabinet File (ISCF): A .cab file used by IS.

Microsoft Cabinet File (MSCF): A .cab file used by Microsoft.
